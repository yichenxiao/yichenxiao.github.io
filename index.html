在Android上搭建Hexo个人博客（本地篇）

下载安装termux
Google Play
F-Droif
更新termux
pkg updata && pkg upgrade -y

查看一下信息，如果OS显示Android 11那么可以完全跟着笔记操作。如果不是，那么跟着笔记操作有的步骤可能会显得多余，但是并不影响结果。(这一步可以跳过）
pkg install neofetch && neofetch

安装git
pkg install git -y
查看版本检查是否安装成功
git --version

安装node长期稳定版
pkg install nodejs-lts -y
惯例查看版本检查是否安装成功
node -v && npm -v

安装proot模拟root
pkg install proot -y
检查是否安装成功
proot --version

模拟root，以后的操作均在此操作前提下进行
termux-chroot
准备工作已经完成，我们正式开始与hexo的交锋，安装hexo框架
npm install -g hexo-cli
检查是否安装成功
hexo -v

将一个目录初始化为网站根目录
hexo init 自己换一个

进入初始化的目录
cd 自定义名字

安装基本工具
npm install

好了，我们已经成功搭建了hexo，现在我们就可以启动本地服务器了。首先，我们先生成静态文件
hexo generate或者hexo g
启动本地服务器
hexo server

此时你会得到一个网址，这就是你的网站地址，复制到浏览器打开就能访问自己的Hexo博客了

在此处你可以找到你创建的文章，以方便你对其进行修改
